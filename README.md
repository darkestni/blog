# Blog

个人博客系统

## 项目简介

一个基于 HTML/CSS/JavaScript 的个人博客网站，支持文章展示、分类浏览等功能。

## 目录结构

```
.
├── blog.html          # 博客列表页
├── article.html       # 文章详情页
├── process_demo.c     # 处理演示代码
├── .github/           # GitHub 配置
└── .gitignore         # Git 忽略文件
```

## 主要页面

- **blog.html** - 博客文章列表页面
- **article.html** - 单篇文章展示页面

## 技术栈

- HTML5
- CSS3
- JavaScript

## 使用方法

1. 克隆本仓库
2. 直接在浏览器中打开 `blog.html` 即可查看
